# Smart Foyer

A smart home management system for controlling and monitoring foyer/entrance devices and automation.

## About

**Smart Foyer** is an intelligent home entrance management application that provides automation and control for foyer-related devices such as lighting, door locks, motion sensors, and security cameras.

## Features

- Automated lighting control based on motion detection
- Smart door lock management
- Real-time sensor monitoring (temperature, humidity, motion)
- Security camera integration
- User-friendly dashboard interface
- Scheduling and automation rules
- Notification system for events and alerts

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdellatifzidane/smart-foyer.git
   cd smart-foyer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
